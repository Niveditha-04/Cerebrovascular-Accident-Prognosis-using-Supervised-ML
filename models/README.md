# Models

The trained Random Forest model (`model.pickle`) is generated automatically when you run the notebook or script.

After running `notebooks/cva_prognosis.ipynb` or `src/cva_prognosis.py`, the file `model.pickle` will appear in this folder.

It is not included in the repository because it requires the dataset to be present first.

## How to generate it

1. Download the dataset (see `data/README.md`)
2. Run the notebook or script
3. `model.pickle` will be saved here automatically
